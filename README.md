# PowerCo Customer Churn Analysis

## 🎯 Objective
To analyze customer churn behavior and determine whether price sensitivity is a key driver of churn using data analysis and machine learning techniques.

---

## 📂 Datasets Used
- **client_data.csv** → Customer demographics, usage, and churn status  
- **price_data.csv** → Historical pricing information  
- **data_description.pdf** → Variable definitions and metadata  

---

## 🔍 Project Workflow

### 1. Data Loading & Cleaning
- Imported datasets using Pandas  
- Converted date columns to datetime format  
- Checked missing values and data types  

### 2. Exploratory Data Analysis (EDA)
- Analyzed churn distribution  
- Studied consumption and pricing patterns  
- Identified trends influencing churn behavior  

### 3. Data Visualization
- Histograms for numerical variables  
- Bar plot for churn distribution  
- Price variation analysis  

### 4. Feature Engineering (if applied)
- Contract duration  
- Price sensitivity features  
- Consumption change metrics  

---

## 📊 Key Insights
- Price sensitivity has a strong relationship with churn  
- Churned customers show different consumption behavior  
- Certain customer segments are more likely to churn based on pricing patterns
